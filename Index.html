🚀 Авторизация через VK Community Messages

Как это работает:

  1. На сайте появляется виджет чата ВК (внизу справа)
  2. Пользователь нажимает на виджет → авторизуется в VK (если не авторизован)
  3. Пользователь пишет первое сообщение или нажимает "Start"
  4. Вы получаете его user_id в событии message_new через Callback API
  5. Сохраняете user_id в БД и можете отправлять сообщения через API
⚠️ Что нужно настроить:
  1. В сообществе VK включить "Сообщения сообщества" и "Возможности бота"
  2. Подключить Callback API или LongPoll для получения событий
  3. Заменить GROUP_ID в коде ниже на ID вашего сообщества

Код виджета (уже встроен на этой странице):

<div id="vk_community_messages"></div>

<script src="https://vk.com/js/api/openapi.js?169"></script>
<script>
  VK.Widgets.CommunityMessages(
    "vk_community_messages",
    {
      tooltipButtonText: "Написать в VK",
      expanded: 0
    },
    GROUP_ID  // Замените на ID вашего сообщества
  );
</script>

✅ Плюсы этого метода:

  • Не нужно создавать VK приложение
  • Официальный виджет от VK
  • Автоматическая авторизация пользователя
  • Простая интеграция (просто вставьте код)
  • Получаете user_id через Callback API

⚠️ Минус:

  • Пользователь должен написать первое сообщение или нажать "Start"

Пример обработки на сервере (Callback API):

// Когда пользователь напишет сообщение, вы получите событие:
{
  "type": "message_new",
  "object": {
    "message": {
      "from_id": 123456789,  // ← Это user_id пользователя!
      "text": "Привет",
      "peer_id": 2000000001
    }
  }
}

// Сохраните from_id в БД и используйте для отправки сообщений
